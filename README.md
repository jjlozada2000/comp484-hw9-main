I think that part 1 was the easiest for me, since displaying then date is pretty simple after figuring out that JavaScript months start at 0 instead of 1.  The hardest for me, though, was part 2 since I had to individually keep track of four separate variables while making sure that each one was being handled without loops.  In this project, I found out that the Date has separate methods for each part of the date, and for the Number I learned that it does not display an error when it's unable to convert something, instead it just returns NaN. Lastly, I learned that .textContent is good for plain text, but .innerHTML is good for slightly more complicated HTML since you can include tags.
